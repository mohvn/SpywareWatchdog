Netsurf

De acordo com seu site: "O NetSurf é um navegador web multi-plataforma para RISC OS, plataformas do tipo UNIX (incluindo Linux), Mac OS X e muito mais".[1] Versão testada: Netsurf 3.9.

Nível de Spyware: Baixo

Ao iniciar o NetSurf faz uma solicitação para obter o ícone do mecanismo de pesquisa padrão, esse me canismo de pesquisa padrão é o Google. Isso foi testado com mitmproxy. Fora isso não há requisições não solicitadas. Depois de seguir o guia de mitigação, este software é classificado como Não é Spyware.
