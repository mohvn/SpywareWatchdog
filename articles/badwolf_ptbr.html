BadWolf

Badwolf é um navegador web minimalista e orientado para a privacidade baseado no WebKitGTK.

Nível de Spyware: Não é Spyware

BadWolf não faz nenhuma requisição não solicitada.

BadWolf é um navegador bastante decente que vale a pena dar uma olhada. Alguns recursos interessantes que valem a pena notar é que o BadWolf possui um switch Javascript e um switch de visualização de imagens, o que é útil para tarefas diárias.
