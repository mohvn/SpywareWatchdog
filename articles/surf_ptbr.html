Surf

De acordo com seu site: "surf é um navegador web simples baseado em WebKit2/GTK+. Ele é capaz de exibir sites e seguir links."[1] Programa testado: v2.0 para Linux. Mitmproxy foi usado para verificar conexões.

Nível de Spyware: Não é Spyware

Surf não faz nenhuma requisição não solicitada. Também é um software totalmente gratuito sob a licença de expat. Foi testado em conjunto com o tabbed, outro software desenvolvido pelas mesmas pessoas para uso com o surf, ele adiciona suporte para abas. Do ponto de vista da privacidade, este navegador é uma excelente escolha.

O Surf requer proxychains para se conectar ao Tor, pois ele suporta apenas proxies HTTP, não SOCKS (que é o que o Tor usa).
