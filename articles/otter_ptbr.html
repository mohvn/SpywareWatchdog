Otter Browser

De acordo com seu site: "Otter Browser visa recriar os melhores aspectos da interface clássica do Opera (12.x) usando o Qt5." Seu lema é: "Controlado pelo usuário, não vice-versa." Versão testada: 0.9.12 (SlackBuild de slackbuilds.org). Programa usado para testar solicitações: Mitmproxy.

Nível de Spyware: Não é Spyware

Otter Browser não faz nenhuma requisição não solicitada. É totalmente código aberto. Os desenvolvedores também não planejam incluir nenhum "recurso" de spyware no futuro. Este parece ser um verdadeiro navegador web focado em privacidade (pelo menos por enquanto).
