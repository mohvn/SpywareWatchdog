Qutebrowser

Qutebrowser é um navegador focado em gestos de teclado com uma GUI simplificada. É baseado em Python e PyQt5 e software livre, licenciado sob a GPL. Programa testado: v1.6.1 para Debian Buster. Mitmproxy foi usado para verificar conexões.

Nível de Spyware: Não é Spyware

Qutebrowser não faz nenhuma requisição não solicitada. Também é um software livre. Este navegador Web é uma ótima opção para usar e não há nada a reclamar do ponto de vista da privacidade. (embora eu realmente não saiba como usar a Interface de Usuário tão bem...) Até agora, esse navegador parece que pode se destacar entre os outros navegadores Web que respeitam a privacidade por aí.
