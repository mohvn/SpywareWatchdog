GNU IceCat

GNU IceCat é um navegador Web que é um fork do Firefox.

Nível de Spyware: Não é Spyware

GNU IceCat é um fork do Firefox que é mais privado e seguro que o Firefox e contém vários recursos de proteção de privacidade. O IceCat 60 não faz conexões não solicitadas quando você o executa. As versões anteriores tinham problemas de privacidade, mas a versão 60 não tem esses problemas. Você pode ler sobre a versão anterior aqui: Revisão do IceCat 59

Recursos de privacidade do IceCat

Do gnu.org:

  • LibreJS: GNU LibreJS visa resolver o problema de JavaScript descrito no artigo de Richard Stallman, The Javascript Trap.
  • Https-Everywhere: Extensão que criptografa suas comunicações com muitos dos principais sites, tornando sua navegação mais segura.
  • AboutIceCat: Adiciona uma página inicial personalizada "about:icecat" com links para informações sobre o software gratuito e recursos de privacidade no IceCat, e caixas de seleção para habilitar e desabilitar aqueles mais propensos a quebrar sites.
  • Contramedidas de Fingerprinting: O Fingerprinting é uma série de técnicas que permitem identificar exclusivamente um navegador com base em características específicas dessa instância específica (como quais fontes estão disponíveis nessa máquina). Ao contrário dos cookies, o usuário não pode optar por não ser rastreado dessa maneira, portanto, o navegador deve evitar dar esse tipo de informação.
